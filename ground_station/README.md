# GROUND STATION CONTROL CENTER [YER KONTROL]

This directory is designated for the Ground Control Station (GCS) GUI application.
Currently, the mission is controlled via the CLI in `mission_control.py`.

## Future Roadmap
- [ ] **Qt/PySide6 Dashboard:** A graphical interface for real-time plotting.
- [ ] **Map Integration:** Live GPS tracking on OpenStreetMaps.
- [ ] **Antenna Tracker:** Scripts to control the servo-based antenna tracker.
